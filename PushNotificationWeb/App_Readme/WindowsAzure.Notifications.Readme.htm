Push Notification Registration Cloud Service

Introduction

The Push Notification Registration Cloud Service lets client devices to register (and unregister) for receiving push notification messages. This package includes a class library, and adds a WebActivator enabled class to configure the service. By default, this service allows anonymous access and uses Windows Azure Storage Tables to store the information of the registered devices.

Prerequisites

  • Windows Azure SDK and Tools
  • NuGet 1.5 or above

Configuration

Configuration your service settings

You can change the default configuration by editing the NotificationService.cs file in the App_Start folder.
